Where the secrets live

Production API keys for Stripe, track1099, and Plaid live in the Cloudflare Worker's secret store — NOT in the Google Sheet. Setting them is a one-time shell command from the agency owner's machine:

cd workers/api
wrangler secret put STRIPE_API_KEY          # sk_test_... or sk_live_...
wrangler secret put STRIPE_WEBHOOK_SECRET   # whsec_...
wrangler secret put TRACK1099_API_KEY
wrangler secret put PLAID_CLIENT_ID
wrangler secret put PLAID_SECRET
