Features

Every modality, every party, every billing model. One tool.

Nine feature areas, each with a one-screenshot summary and a "how it works" walkthrough. ASL videos coming online with the design-partner cohort in Phase 0.

Scheduling

Six dashboards, one filter bar. URL-persisted state. Smart-fill with score breakdown. Cancellation modal with live tier preview.

Interpreter app

Billing

Five consolidation modes per client. Monotonic invoice numbers. Payout PDF with Labor + Expenses tables. QuickBooks, Xero, NetSuite, Bill.com.

Document translation

Human-in-the-loop. DeepL Pro where it supports the pair; Claude elsewhere. No pre-fill on medical or legal.

AI intake
